# 🌸 Myreva Olfactives — Inventory & Contact Page

> *Where Scent Becomes Spell.*

A single-page website for **Myreva Olfactives** — a Pakistani fragrance brand offering premium impression perfumes. The page displays the full product pricelist with colour-coded categories and direct contact links for ordering.

🌐 **Live Site:** [myrevaolfactives.netlify.app](https://myrevaolfactives.netlify.app)

---

## ✨ Features

- **Fragrance Inventory Table** – Lists all available perfumes with article name, inspiration, colour-coded category pill (Homme / Femme / Unisex), and pricing for both 55ml and 3ml sizes
- **Category Legend** – Colour-coded pills with a Homme / Femme / Unisex legend at the bottom of the table
- **Animated Hero Section** – Letter-by-letter logo reveal, gold dust particle effect, and a smooth parallax scroll on the hero text
- **Scroll-Reveal Animations** – Table rows animate in sequentially as they enter the viewport
- **Contact Section** – Direct links to Facebook, Instagram, WhatsApp, and Email with hover animations
- **Smooth Scroll Navigation** – Anchor buttons between the hero, pricelist, and contact sections
- **Fully Responsive** – Adapts cleanly to mobile and desktop; table columns reflow on small screens
- **No frameworks, no build step** – Pure HTML with embedded CSS and JS; opens straight in a browser
